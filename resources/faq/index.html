Filtering Basics

To see the filters that have been applied:

  1. Go to the top of the dashboard
  2. Look for the filter display area where active filters are shown
  3. Each active filter will be displayed with its criteria

Active filters are displayed at the top of the dashboard

To remove filters:

  1. Go to the top of the dashboard where filters are displayed
  2. Find the filter you want to remove
  3. Click the "x" button next to the filter
  4. The filter will be removed and the dashboard will update automatically

Click the "x" button to remove individual filters

Author Filtering

To filter by author directly from the dashboard:

  1. Hover your mouse over an author's name in the data visualization
  2. You'll see interactive options appear
  3. Click the "+" button if you want to include this author in your search
  4. Click the "-" button if you want to exclude this author from your search
  5. The dashboard will update automatically with your filter applied

Hover over an author name to reveal the + and - filter buttons

To search for specific authors using the advanced filter interface:

  1. Go to the top of the dashboard and click the blue "+" button
  2. Click the "Select a field" dropdown
  3. Type "author" to find the author field
  4. Click "author" from the dropdown options
  5. Click the "Select operator" field
  6. Select "is" as the operator
  7. Click the "Enter a value" field
  8. Type the name of the author you're looking for
  9. Add "*" before or after the string as a wildcard if necessary (Example: *Tyler*)
  10. Note that the search is case sensitive
  11. Click "Add filter"
  12. Check the authors table to verify only matching authors are displayed

Tip: Use wildcards (*) to find partial matches. For example, "*Smith*" will find "John Smith", "Mary Smith-Jones", etc.

Affiliation Filtering

Filtering by affiliation works similarly to author filtering:

  1. Hover your mouse over an affiliation name in the data visualization
  2. Interactive options will appear
  3. Click the "+" button to include this affiliation in your search
  4. Click the "-" button to exclude this affiliation from your search
  5. The dashboard will update to show only data matching your affiliation criteria

Interactive filtering works the same for both authors and affiliations

View your applied affiliation filters at the top of the dashboard

Note: The same hover-and-click mechanism used for authors applies to affiliations. Look for + and - buttons when you hover over affiliation names in the visualization.

To search for specific affiliations using the advanced filter interface:

  1. Click the blue "+" button at the top of the dashboard
  2. Select "affiliation" from the field dropdown instead of "author"
  3. Choose your operator (typically "is" or "contains")
  4. Enter the affiliation name (use wildcards like *University* for broader matches)
  5. Click "Add filter" to apply

Click the "+" button to start creating an affiliation filter

Type "affiliation" to find the affiliation field option

Advanced Search Features

Yes! The dashboard supports advanced logical operations:

  • AND logic: Apply multiple filters to narrow down results (e.g., show papers by "Author A" AND from "University B")
  • OR logic: Use OR operators to broaden your search (e.g., show papers by "Author A" OR "Author B")
  • Combination filters: You can combine multiple filters with different logical operators
  • Remove filters: Each filter can be removed individually without affecting others

Choose logical operators like "is", "contains", or "is not"

Multiple filters work together using AND/OR logic

Pro Tip: Experiment with different combinations of filters to find the exact data you're looking for!

The dashboard supports the asterisk (*) wildcard character:

  • * (asterisk) - Matches any number of characters
  • Examples:
    • *Tyler* - Finds any name containing "Tyler"
    • Smith* - Finds names starting with "Smith"
    • *University - Finds affiliations ending with "University"

Example of using wildcards (*) in search values for broader matches

Important: Searches are case-sensitive, so "tyler" and "Tyler" will return different results.

  4. Use the filter display area to monitor your active filters
  5. Remove or modify filters as needed to refine your search

Filter Management

After applying filters, you can verify they're working by:

  1. Check the data tables: Look at the authors table or other data displays to ensure only matching results are shown
  2. Review active filters: Go to the top of the dashboard to see all currently applied filters
  3. Monitor data counts: Notice how the number of results changes when you add or remove filters
  4. Test filter combinations: Try different combinations to ensure the logic is working as expected

If filters aren't working as expected, try these troubleshooting steps:

  1. Refresh the page: Sometimes a simple page refresh resolves display issues
  2. Clear all filters: Remove all filters and start over to ensure a clean state
  3. Check filter syntax: Ensure you're using the correct case and wildcard characters
  4. Wait for loading: The dashboard may need a moment to process complex filter combinations
  5. Check your internet connection: Ensure you have a stable connection to the server
