Democratizing Data: Making Public Data Assets More Discoverable

We show how public data are being used in science so that the government can make more transparent public investments. Our platform uses machine learning and natural language processing to enable government agencies and researchers to quickly find the information they need.

Our Mission & Vision

Why We Exist

Research is the foundation of evidence-based policymaking. Federal agencies invest billions of dollars annually in collecting, curating, and maintaining high-quality datasets that support scientific research and inform critical policy decisions. However, understanding how these data assets are actually used in research has remained a significant challenge.

Our platform addresses this gap by providing comprehensive insights into how federal data assets are referenced and utilized in published research. We help agencies demonstrate the value and impact of their data investments while enabling researchers to discover trusted datasets for their work.

Our Approach to Data Discovery

We employ a sophisticated three-stage workflow that combines cutting-edge machine learning with human validation to ensure accuracy and reliability:

Key Objectives

  • Demonstrate the value of public data investments
  • Enable evidence-based policy making
  • Improve data discovery for researchers
  • Foster transparent government data use
  • Build trusted data infrastructures

Our Technical Methodology

This project is made possible through the use of advanced machine learning algorithms combined with rigorous human validation. Our approach ensures that results are both accurate and reliable, providing agencies and researchers with trustworthy insights into data usage patterns.

Machine Learning Algorithms
Three winning models built on machine learning and natural language processing tools for dataset identification

Data Validation
Human reviewers validate ML output using publication snippets to ensure accuracy and reliability

Machine Learning Models

Our system combines three complementary approaches developed through the Kaggle "Show Us the Data" competition:

Model 1: Deep Learning - Sentence Context

A deep learning approach that learns what types of sentences contain dataset references. Most robust to new datasets and evaluates all document text.

Model 2: Deep Learning - Entity Names

Extracts entity names from text and classifies whether an entity represents a dataset or not.

Model 3: Pattern Matching

A rule-based approach that searches for patterns similar to existing known datasets.

Data Validation Process

Our validation tool provides reviewers with publication snippets containing candidate dataset references identified by our models. Validators determine whether these references correctly identify the target datasets.

Validation Results Provide:

  1. Model Performance Insights - Understanding how well models identify datasets across different domains
  2. Related Dataset Discovery - Finding other datasets used in conjunction with agency data
  3. Usage Pattern Analysis - Identifying how researchers utilize agency data in their work
