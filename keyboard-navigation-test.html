Keyboard Navigation Test - Democratizing Data

Testing Instructions

Use only the keyboard for this test:

  • Tab - Move forward through interactive elements
  • Shift + Tab - Move backward through interactive elements
  • Enter - Activate links and buttons
  • Space - Activate buttons and checkboxes
  • Arrow Keys - Navigate within dropdown menus
  • Escape - Close dropdown menus and modals

Check for: Visible focus indicators, logical tab order, all functionality accessible via keyboard

1. Homepage Navigation Test

Open Homepage
  • Logo is focusable and activates with Enter
  • Main navigation items are focusable in logical order
  • Dropdown menus open with Enter/Space and close with Escape
  • Dropdown items are navigable with arrow keys
  • All links in content sections are focusable
  • "Load More" buttons (if present) are focusable and functional
  • Footer links are focusable in logical order
  • Focus indicators are clearly visible on all elements

2. About Pages Test

About Overview Team Contact Us
  • Breadcrumb navigation is focusable and functional
  • All content links are keyboard accessible
  • Contact form (if present) is fully keyboard navigable
  • Form fields can be filled using only keyboard
  • Form submission works with Enter key

3. Tools Pages Test

Tools Index USDA Dashboard Workforce Dashboard
  • All tool links and buttons are keyboard accessible
  • External links (GitHub) open correctly with Enter
  • Dashboard navigation elements are focusable
  • Any interactive charts/widgets are keyboard accessible

4. Events Page Test

Events Page
  • Event cards are focusable and navigable
  • "Load More Events" button is keyboard accessible
  • Event registration links work with keyboard
  • Past events section is keyboard navigable

5. Resources Page Test

Resources Index Additional Resources
  • Resource cards and links are keyboard accessible
  • "Load More Publications" button works with keyboard
  • PDF download links are keyboard accessible
  • External resource links work correctly

6. Focus Management Test

  • Focus indicators are visible and high contrast
  • Focus doesn't get trapped in any section
  • Tab order is logical and intuitive
  • No keyboard traps exist
  • Focus returns to appropriate element after modal/dropdown closes
  • Skip links are present and functional (if implemented)

7. Mobile Navigation Test

Resize browser to mobile width and test:

  • Mobile menu toggle button is keyboard accessible
  • Mobile menu opens/closes with Enter/Space
  • Mobile menu items are keyboard navigable
  • Mobile dropdown menus work with keyboard
